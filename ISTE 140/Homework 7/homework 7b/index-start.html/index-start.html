Hello World

In previous semesters I noticed students struggle to make a media query correctly in their projects. To hopefully alleviate this I want everyone to submit a file with a simple media query that changes a property when a certain width (or breakpoint) is reached.  I am not looking for you to do something big/complex/time consuming, just merely demonstrate that you can write a simple media query, then in the projects you can ramp it up.

The Assignment:

1.  Use the starter code I attached that has an <h1> that says Hello World at the top.

2. Add css in the <head> style attribute (an embedded style sheet). It isn't neccessary to do an externally linked stylesheet for this HW. this way you can just submit a single index.html in the dropbox which contains all of the code (css and html in one). The CSS should achieve the following:

  • When the viewport (width of page in your browser) is greater than 800px make the text color of the "Hello World" in the h1 red.
  • When the viewport is less than 800px make the text color blue.

There are a couple ways of doing this, but you essentially need to create @media rule. Review our examples or use search to find other examples if needed. You can achieve this using either the min-width or max-width rule.

I don't think this hw should be all to time consuming, but if you are stuck please message me on Slack or set up office hours with me and I will be glad to help.

When completed submit the index.html to dropbox.
