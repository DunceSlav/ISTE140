What is Bitcoin?

Bitcoin was the first decentralized digital currency.

How do you get Bitcoin?

There are two ways of getting bitcoin: buying and mining. However, before you buy or mine any bitcoin, you need to get a cryptocurrency wallet. You can set up a wallet at coinbase.com

How to buy Bitcoin

You can buy Bitcoin from many different places. One place you can buy Bitcoin and almost any other form of cryptocurrency without any fee is robinhood.com. Another place you can buy Bitcoin is the same place you made your cryptocurrency wallet: coinbase.com.

How to mine Bitcoin

Mining is the term used to describe the process that verifies cryptocurrency transactions. At the end of every verification there is a small reward for doing the work. This verification can be done by anybody that has asccess to a computer. There are two ways to mine any cryptocurrency like Bitcoin: with the computer's GPU or an ASIC machine. The ASIC machine is a machine that was designed specifically to mine cryptocurrency and is a lot more effective at mining than a GPU, however, they are much more expensive. An easy application to use and set up to start mining is nicehash.com.
